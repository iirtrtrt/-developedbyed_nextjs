## Getting Started

### Prerequisites

Recommended versions:<br>
node18 LTS<br>
next13<br>
react18<br>

.env file format below:<br>
DATABASE_URL=<br>
GOOGLE_CLIENT_ID=<br>
GOOGLE_CLIENT_SECRET=<br>
AUTH_SECRET=<br>

## Installation

### Localhost

1. Download:

```sh
git clone https://github.com/iirtrtrt/developedbyed_nextjs.git
```

2. Go to the root directory:

```sh
cd ./cv3_nextjs
```

## Run

### Localhost

1. Install packages:

```sh
npm install
```

2. Run server:

```sh
npm run build
npm run start
```
